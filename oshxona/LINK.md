# Taste of Uzbekistan – platforma linklari

## Lokal (kompyuteringizda)

Server ishlaganda brauzerda oching:

**http://localhost:4000**

- Asosiy sahifa: http://localhost:4000  
- Admin panel: http://localhost:4000/admin.html  

Serverni ishga tushirish: `npm start`

---

## Netlify (internetda)

Saytni Netlify’ga deploy qilganingizdan keyin link shu ko‘rinishda bo‘ladi:

**https://SITENAME.netlify.app**

- **SITENAME** o‘rniga Netlify sizga bergan nom yoki o‘zingiz tanlagan nom keladi.

Linkni olish uchun:
1. https://app.netlify.com ga kiring
2. **Add new site** → **Deploy manually**
3. `oshxona` papkasini (yoki index.html, admin.html, styles.css, not.js, admin.js, 404.html, sitemap.xml, robots.txt) tortib tashlang
4. Deploy tugagach, **Site URL** / **Website URL** – o‘sha sizning platforma linki

Batafsil: **NETLIFY-DEPLOY.md** faylida.
